Banco Nación (BNA+)

Aplicación oficial del Banco Nación para consultar saldo, transferir dinero y pagar servicios de manera rápida y segura.

Funciones principales

  • Ver el saldo de tus cuentas bancarias.
  • Transferir dinero usando CBU o alias.
  • Pagar servicios escaneando el código de la boleta.

Consejos útiles

✔ Guardá los destinatarios frecuentes para transferir más rápido.

✔ Revisá siempre el comprobante después de cada pago.

✔ Mantené la app actualizada para mayor seguridad.
